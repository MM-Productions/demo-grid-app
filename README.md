# QR Grid App

Интерактивное React-приложение с гридом изображений и QR-кодами для планшета.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите в режиме разработки:
```bash
npm run dev
```

3. Соберите для продакшена:
```bash
npm run build
```

## Добавление изображений

1. Положите ваши изображения в:
   - `src/assets/images/` - основные картинки (назовите их image1.jpg, image2.jpg, и т.д.)
   - `src/assets/qrcodes/` - QR-коды (назовите их qr1.png, qr2.png, и т.д.)

2. Обновите пути в `src/App.jsx` в массиве `gridItems`

## Анимации

Каждое изображение использует свою уникальную анимацию:
1. **flip** - 3D переворот
2. **zoom-fade** - увеличение с затуханием
3. **slide-up** - выезд снизу
4. **ripple** - волновое расширение
5. **rotate-scale** - вращение и масштабирование
6. **grid-split** - разделение грида
7. **fade-cross** - перекрестное затухание
8. **bounce-scale** - упругое масштабирование
9. **diagonal-wipe** - диагональная шторка
10. **blur-focus** - размытие в фокус
